# good-techstack

你是否有着絕妙的點子，但苦於少了最後一塊拼圖，難以實現完整的產品？那你需要這套。一行指令。直接完成。

---

你有一個點子，想要把它做出來、放到網路上、看看會不會成。

但你不會寫程式，也沒學過。你只知道這個東西做出來應該會有人用。

這組工具是為你準備的。

你不用從零開始學程式。你只需要學會打開終端機、輸入一行指令，剩下的人工智慧助手會帶你走完。

---

## 展示用的樣品與給客人購買的產品是不同的

你用人工智慧隨便玩玩，請它幫你生一個畫面，打開瀏覽器看起來沒問題。

然後你想要真的上線給別人用。

問題來了：沒有地方存資料、沒有登入功能、沒有地方可以放你的網站。玩玩的時候只有一個畫面，但真正的產品需要會員系統、需要存資料、需要處理錯誤、需要一個網址讓別人連進來——這些是玩玩的時候從來不用想的。

Good-techstack 就是來補這個洞的。它幫你把那些麻煩的底層全部處理好，讓你專心想你的點子。

---

## 開始之前，你需要準備的

你只需要兩樣東西：

1. **一臺電腦**
2. **一個點子**

不需要信用卡。不需要學過軟體工程。不需要知道什麼是伺服器。

---

## 第一步：打開終端機

如果你用的不是蘋果電腦，請照著做：

**按下鍵盤上的 `Win` + `R`**

跳出一個小視窗，在裡面輸入：

```
cmd
```

然後按下 `Enter`（回車鍵）。

你會看到一個黑色的視窗。這就是我們要的終端機。

---

## 第二步：安裝 WSL（Windows 的 Linux 子系統）

在黑色視窗裡，輸入這行指令，然後按 `Enter`：

```
wsl --install
```

它會開始下載安裝 Linux 系統。這個過程需要一些時間，你可以去喝杯水。

**安裝完成後，重新啟動你的電腦。**

---

## 第三步：啟動 WSL

重開機之後，按下 `Win` 鍵，在搜尋欄輸入 `wsl`，然後按 `Enter`。

你會看到一個新的黑色視窗，裡面有類似這樣的文字：

```
user@computer:~$
```

這表示你現在在 Linux 環境裡了。

---

## 第四步：輸入這行指令

在 WSL 視窗裡，輸入這行指令，然後按 `Enter`：

```bash
curl --proto '=https' --tlsv1.2 -fsSL https://raw.githubusercontent.com/eouzoe/good-techstack/main/scripts/start.sh | bash
```

它會開始下載需要的東西。

---

## 接下來會發生什麼

這行指令做完之後，會自動啟動一個人工智慧助手。這個助手會：

1. 引導你建立 Cloudflare 免費帳號（不用信用卡）
2. 設定開發環境
3. 跟你聊天，了解你的產品點子
4. 幫你產生程式碼——資料庫、後端、前端，全部串好
5. 部署到網路上，給你一個網址

你只需要回答它的問題，告訴它你想做什麼。

---

## 你的人工智慧助手

這是改變一切的關鍵。

大部分的開發工具是設計給人看的——說明書、文件、教學。Good-techstack 同時設計給你和你的助手。

你的助手會讀這些文件——刻意為它寫的——然後它就知道了：

- **有哪些工具可以用**，以及該用哪個。它不會亂建議一些不屬於這組工具的東西。
- **該裝哪個版本**。它會先確認，不會用猜的。
- **該怎麼寫程式**，以及該避開什麼。每個常見的錯誤都有記錄。
- **怎麼確認自己寫對了**。它有一套檢查程序，全部通過才算完成。

結果就是：你的助手每次都能產出真正能動的程式。不是那種看起來好像對、一跑就壞掉的東西。

當你跟它說「幫我加一個登入頁面」，你得到的就是一個登入頁面——有登入狀態管理、有密碼重設、有第三方登入。

當你跟它說「上線」，它就幫你部署了。資料庫建好了、環境設定好了、服務跑起來了。

## 這組工具包含了什麼

| 項目 | 說明 |
|------|------|
| 跑程式的引擎 | Bun — 跑得很快 |
| 後端框架 | Hono + oRPC — 讓前後端資料互通 |
| 資料庫 | Cloudflare D1 — 免管理的資料庫 |
| 登入系統 | better-auth — 信箱註冊、第三方登入，都包了 |
| 前端框架 | TanStack Start — 快速的前端畫面 |
| 界面元件 | shadcn/ui — 好看又好用的元件庫 |
| 程式碼檢查 | oxlint — 自動檢查程式有沒有寫錯 |
| 部署平台 | Cloudflare Workers — 有使用者之前都免費 |
| 開發環境 | Nix — 所有人的環境都一樣，不會出現「我電腦可以跑」的問題 |
