## Drizzle ORM

**Version**: 0.45.2 stable (v1.0.0-rc.4 pre-release)

### 角色

資料庫存取層，運行在 Workers/D1 上：

```ts
import { drizzle } from "drizzle-orm/d1";
const db = drizzle(env.DB);
const result = await db.select().from(items).where(eq(items.userId, userId));
```

### 與 better-auth 共存

| 角色      | 工具                                           |
| --------- | ---------------------------------------------- |
| Auth 表格 | better-auth 自動管理，不需 schema              |
| 應用表格  | Drizzle ORM 手動定義                           |
| Migration | Drizzle Kit 產生 → wrangler d1 migrations 執行 |

### 為何選 Drizzle 非 Prisma

| 面向       | Drizzle                     | Prisma                    |
| ---------- | --------------------------- | ------------------------- |
| Runtime    | 0 overhead (SQL-like)       | 厚 runtime binary         |
| Workers    | `drizzle-orm/d1` 原生       | `@prisma/adapter-d1` beta |
| Bundle     | ~10KB                       | ~5MB                      |
| Agent 友善 | SQL-like, AI 易生成正確語句 | Schema DSL 需學習         |

### Bun:sqlite 記憶體測試

```ts
import { Database } from "bun:sqlite";
import { drizzle } from "drizzle-orm/bun-sqlite";
const sqlite = new Database(":memory:");
const db = drizzle(sqlite);
```

不需 mock、不需 Docker。

## D1 (Cloudflare SQLite)

- Serverless SQLite, 5GB 免費
- Migration: `wrangler d1 migrations`
- Binding: `env.DB`
- 搬遷：dump → import (標準 SQLite)

## 型別安全整合

```
Zod schema (packages/contract/)
  → Drizzle: z.infer<> 對應 DB row type
  → oRPC: .input(UserCreate).output(UserResponse)
  → TanStack Form: zodResolver(UserCreate)
```
